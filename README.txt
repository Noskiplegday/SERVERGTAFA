SA-MP Streamer Plugin
=====================

Đây là tài liệu ngắn của plugin Streamer cho SA-MP/open.mp.

Plugin này dùng để stream các thành phần trong game như object, pickup,
checkpoint, race checkpoint, map icon, 3D text label và actor theo tick do
server quy định. Plugin cũng hỗ trợ phát hiện khu vực cơ bản.

Vì được viết bằng C++, plugin giảm được nhiều chi phí xử lý so với việc tự
stream hoàn toàn bằng Pawn. Nhờ vậy, Streamer Plugin thường nhanh và nhẹ hơn
các cách triển khai streamer thuần Pawn.
